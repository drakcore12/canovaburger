# Voz y mensajes - Canova Burgers

## Tono
Directo, seguro, cercano, provocador, divertido y apetitoso. Nunca corporativo, delicado, infantil o excesivamente gourmet.

## Fórmula de copy
1. Abrir con una sensación o un beneficio.
2. Usar frases breves.
3. Mostrar producto real.
4. Cerrar con acción clara.

## Palabras propias
Smash, double, melt, loaded, crispy, juicy, fire, crave, hungry, golden, stack, antojo, fundido, crujiente y jugoso.

## Palabras a evitar
Gourmet, delicado, exclusivo, light, dietético, culpa, pecado y refinado.
